# Privacy and Sanitization Notes

This workspace follows a conservative public-upload policy.

## Kept

- Source code needed to understand and reproduce experiments.
- Lightweight generated examples that do not contain personal information.
- Theory assignment pages only after visual inspection showed no obvious name or student ID.
- Midterm LaTeX source with the author field anonymized.

## Removed or Not Copied

- Submitted PDF reports with cover pages, name, student ID, teacher name, date, and institution metadata.
- DOCX files that may contain hidden metadata.
- Zip/RAR archives and duplicate packaged submissions.
- Textbook PDFs and lecture slides.
- Model weights, checkpoints, full model folders, LoRA weights, downloaded datasets, Python caches, and LaTeX build products.

## Report Replacement Strategy

For private experiment reports, the repository uses README summaries derived from the report content and project source code. This keeps the learning record useful without publishing private covers or full submitted documents.

## Temporary Files

The `tmp/` directory is for local inspection and rendering only. It is ignored by Git.
