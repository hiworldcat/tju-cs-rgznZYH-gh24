# 项目目录

四个课程大作业/实验被整理为相互独立的项目目录。

| 目录 | 主题 | 公开内容 |
| --- | --- | --- |
| `01-eight-queens` | 逻辑推理与约束满足 | Python 求解器 |
| `02-robot-path-planning` | 搜索与路径规划 | Python 实现和少量结果图片 |
| `03-lenet-garbage-classification` | 深度学习与迁移学习 | MindSpore 源码、依赖说明、结果 CSV |
| `04-poetry-ink-diffusion` | 大模型图像生成 | Stable Diffusion 流程脚本、提示词、示例输出 |

带隐私信息的 PDF 报告和提交压缩包未纳入。每个项目目录都包含 README，用来说明实验内容、文件结构和复现方式。
