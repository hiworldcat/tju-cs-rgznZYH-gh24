# 八皇后逻辑推理实验

本项目求解经典八皇后约束满足问题，并比较多种方法：

- 穷举枚举结合逻辑筛选；
- DFS 回溯；
- BFS 搜索；
- kanren 风格逻辑编程，并自定义不等约束和斜线约束。

## 文件

```text
eight_queens.py
```

## 依赖

```bash
pip install kanren
```

原实验主要适配 `kanren 0.2.3`，源码中保留了针对该版本的兼容处理。

## 运行

```bash
python eight_queens.py
```

脚本会输出不同方法的解数量和运行时间对比。报告结论认为，在八皇后这样的小规模约束搜索空间中，DFS/回溯效率最高；逻辑编程虽然运行更慢，但约束表达更声明式，可读性更好。
